# Dofbot AI Controller

AI-powered robot arm controller using natural language — Flask + Groq AI + Raspberry Pi 5

## Files
- `robot_controller_api.py` — Main Flask API
- `robot_web_ui.html` — Web interface
- `movements.json` — 17 verified positions

## Setup
```bash
pip install flask groq
python robot_controller_api.py
```
Then open `robot_web_ui.html` in your browser.
